本产品不提供跨境联网（翻墙）服务，本公司提供的大陆地区IP仅限于大陆地区使用，其它地区的IP资源（除中国大陆地区以外）无法在大陆地区范围内使用，因此产生的相关责任用户自负，对此飞蚁代理不承担任何法律责任。

长效IP：长效IP可以保持长期在线，网络质量好，链接稳定，带宽有保证，支持HTTP/HTTPS/SOCKS5协议，也支持PPTP/L2TP连接，按照线路数量包月计费。

请求分配端口示例：

http://ip.feiyiproxy.com:88/open?user_name=lt_test_1×tamp=15889653664&md5=00112233445566778899AABBCCDDEEFF&pattern=json&number=2

请求各字段含义：

ip.feiyiproxy.com:88 是代理服务器，不同客户可能分配不同服务器，实际使用根据我方提供为准；

user_name 是给用户分配的用户名

timestamp 是当前时间戳，每次不能重复，必须比上次的值大（单位为 ms）

md5 是 md5(user_name+password +timestamp)，timestamp 要转为字符形式。

password 为分配给用户的密码。

pattern 响应数据格式（txt|json）

number 请求分配的代理个数（最小为 1，最大 200）

注：1、此申请方法为获取ip，实际使用ip时为防止其他人盗用，可选择不同验证方法验证使用；验证方法详见末尾说明；

2、客户方测试前期为操作简单，一般会给客户设置提取ip时不验证MD5，保证账号MD5长度一致就可以；

响应： json 格式：

{ "code": 100, "left_ip": 4914, "left_time": 1876270, "number": 2, "domain": "ip.feiyiproxy.com", "port": [ 14343, 14344 ] }

响应各字段含义：

code 为响应代码号

left_ip 为账号上一次申请后剩余IP数

left_time为账号剩余使用时间（单位秒）

number 为本次申请个数

domain 为使用代理的IP（此IP为我方服务器，非实际使用IP）

port 为代理端口号

响应：txt 格式：

响应各字段含义：

第一行为响应状态

第二行为为账号上一次申请后剩余IP数

第三行为账号剩余使用时间（单位秒）

第四行为本次申请IP数量

code 为响应代码号

重置本用户已使用ip

http://ip.feiyiproxy.com:88/reset_ip?user_name=lt_test_1×t amp=15889653664&md5=00112233445566778899AABBCCDDEEFF& pattern=json

重置ip是指重置去重周期内的ip，使去重周期内的ip能再次使用，对ip重复要求严格请勿使用此接口

为防止其他非法使用，用户可根据自身环境选择一种适合自己的验证方式:

1、验证使用ip；只有提取代理时的出口ip和使用代理时的出口ip一致方可正常使用；此方法只能是谁取谁用，如果出口ip一致则可用； 例：在公司出口ip为183.14.133.168通过API申请代理，使用时只能在183.14.133.168为出口的内网中使用代理更换网络环境的话代理将不可用；

2、验证账号名密码 使用时候：在http/https的请求头上添加如下字段： Proxy-Authorization: Basic user_name:password\r\n

3、添加白名单，只有添加了白名单设备才能使用该账号提取的ip，否则无法使用 白名单添加例子：

http://192.168.2.188:88/add_whitelist?user_name=lushanshan×tamp=1530153650&md5=0db7e8f36da9694c2aeebd3f9dd5b65f&pattern=json&user_ip=192.168.2.45 API说明；

192.168.2.88:88为账号所在服务器，使用的哪个服务器就在哪个服务器上添加白名单；

user_name 是为某个账号添加白名单

MD5 (user_name+password +timestamp)，timestamp 要转为字符形式；

user_ip 是你要添加的ip，一次只能添加一个；（白名单为机器的公网ip）
